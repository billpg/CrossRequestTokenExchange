Issuer Demo.

This is the issuer demo. This service will act as the Issuer while yours will act as the Initiator.

Have your service make a POST request according to the description in the project's github to this URL:

https:///Issuer/TokenCall

This request will include your URL to a hash of your JSON request. Please note that if this service detects a large number of failed requests, it will return a 429 error indicating its refusal to participate.

If the request is successful, the response will include a Bearer token. You may validate this token by using it with a GET request tp this URL:

https:///Issuer/TokenTest

If you have any issues, please open a ticket in this project's github.
