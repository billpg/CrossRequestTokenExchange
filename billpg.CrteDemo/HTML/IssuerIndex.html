Issuer Demo.

This is the issuer demo. This server will act as the Issuer while yours will act as the Initiator.

CRTE normally requires a pre-existing relationship between the two participants. As that won't be possible for this demo, please set up a billpg-com-crte.json file on your website before starting.

Once that file is in place, we need to know the URL to send the TokenIssue POST requests to. Fill this in here and click the Start button. The next page will return the URL on this server for you to send your POST request to.
