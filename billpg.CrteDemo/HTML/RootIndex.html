Cross Request Token Exchange

This website implements each side of the exchange, in a way that you can test against your own web service.

See this project's github for a full description of the exchange.

Issuer

This section is for your service to act as the Initiator and this service will act as the Issuer. You code will make a PPOST request to thi service and we'll respond by making a separate POST request back to your service with the newly issued Bearer token. There's a separate end-point that will test you have a valid Bearer token.
